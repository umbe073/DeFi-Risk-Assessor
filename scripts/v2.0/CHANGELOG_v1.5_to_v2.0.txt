#### Unified App Icon Support
```bash
BUNDLE_IDENTIFIER="com.defi.riskassessment"
APP_BUNDLE="true"
CFBundleIdentifier="com.defi.riskassessment"
CFBundleName="Token Risk Assessment Tool"
CFBundleDisplayName="Token Risk Assessment Tool"
PARENT_BUNDLE_ID="com.defi.riskassessment"
INHERIT_BUNDLE_ID="com.defi.riskassessment"
```

#### Background Operation
```bash
NSApplicationActivationPolicy="accessory"
LSUIElement="true"
```

#### macOS Compatibility
```bash
TK_SILENCE_DEPRECATION="1"
TK_FORCE_BASIC_MODE="1"
TK_SKIP_ALL_MACOS_CHECKS="1"
TK_DISABLE_ALL_MACOS_FEATURES="1"
```
