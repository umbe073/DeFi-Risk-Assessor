## 📊 SOCIAL DATA CATEGORY SOURCES

### 1. Twitter (X) Integration ⚠️ REQUIRES API KEY
**API Service**: Twitter API v2
**Primary Use**: Sentiment analysis, engagement rates, follower growth, mention frequency
**Compliance Value**: Real-time sentiment, community engagement, influencer mentions

#### 🔑 API Key Requirements:
- **Account Type**: Twitter Developer Account
- **Access Level**: Academic Research (preferred) or Elevated Access
- **Rate Limits**: 
  - Academic: 500K tweets/month, 300 requests/15min
  - Elevated: 2M tweets/month, 300 requests/15min
- **Cost**: Free for Academic Research, $100/month for Elevated Access

#### 📋 Setup Steps:
1. **Create Twitter Developer Account**:
   - Go to https://developer.twitter.com/
   - Sign in with your Twitter account
   - Apply for a developer account
   - Fill out the application form (research/academic purpose)

2. **Create App**:
   - Navigate to "Projects & Apps" → "Create App"
   - Choose "Academic Research" if eligible
   - Fill out the app details and use case

3. **Get API Keys**:
   - API Key (Consumer Key)
   - API Secret (Consumer Secret)
   - Bearer Token (for v2 API)
   - Access Token & Secret (for user authentication)

4. **Environment Variables**:
   ```bash
   export TWITTER_API_KEY="your_api_key"
   export TWITTER_API_SECRET="your_api_secret"
   export TWITTER_BEARER_TOKEN="your_bearer_token"
   ```

#### 🔧 Integration Features:
- **Sentiment Analysis**: Analyze tweet sentiment for tokens
- **Engagement Metrics**: Track likes, retweets, replies
- **Influencer Mentions**: Monitor crypto influencer discussions
- **Trend Analysis**: Track mention frequency over time
- **Community Size**: Monitor follower growth and engagement

### 2. Telegram Integration ⚠️ REQUIRES API KEY
**API Service**: Telegram Bot API
**Primary Use**: Channel member count, message frequency, bot activity
**Compliance Value**: Community size, engagement patterns, suspicious bot activity

#### 🔑 API Key Requirements:
- **Account Type**: Telegram Bot Account
- **Access Level**: Free tier available
- **Rate Limits**: 30 messages/second, 20 requests/second
- **Cost**: Free

#### 📋 Setup Steps:
1. **Create Telegram Bot**:
   - Message @BotFather on Telegram
   - Send `/newbot` command
   - Choose bot name and username
   - Receive bot token

2. **Get Bot Token**:
   - BotFather will provide a token like: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`

3. **Environment Variables**:
   ```bash
   export TELEGRAM_BOT_TOKEN="your_bot_token"
   ```

#### 🔧 Integration Features:
- **Channel Analysis**: Monitor crypto project channels
- **Member Count**: Track community growth
- **Message Frequency**: Analyze engagement patterns
- **Bot Detection**: Identify suspicious bot activity
- **Community Sentiment**: Analyze message content and reactions

### 4. Reddit Integration ⚠️ REQUIRES API KEY
**API Service**: Reddit API
**Primary Use**: Subreddit mentions, sentiment analysis, post frequency
**Compliance Value**: Community sentiment, discussion quality, red flag detection

#### 🔑 API Key Requirements:
- **Account Type**: Reddit Developer Account
- **Access Level**: Free tier available
- **Rate Limits**: 60 requests/minute
- **Cost**: Free

#### 📋 Setup Steps:
1. **Create Reddit App**:
   - Go to https://www.reddit.com/prefs/apps
   - Click "Create App" or "Create Another App"
   - Choose "script" as app type
   - Fill out app details

2. **Get API Credentials**:
   - Client ID (under app name)
   - Client Secret (if available)
   - User Agent (your app name)

3. **Environment Variables**:
   ```bash
   export REDDIT_CLIENT_ID="your_client_id"
   export REDDIT_CLIENT_SECRET="your_client_secret"
   export REDDIT_USER_AGENT="your_app_name/1.0"
   ```

#### 🔧 Integration Features:
- **Subreddit Monitoring**: Track mentions in crypto subreddits
- **Sentiment Analysis**: Analyze post and comment sentiment
- **Discussion Quality**: Assess community discussion depth
- **Red Flag Detection**: Identify suspicious posts or patterns
- **Trend Analysis**: Monitor mention frequency over time
